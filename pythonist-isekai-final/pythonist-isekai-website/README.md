# Reincarnated as a Pythonist

A purple/black, anime-isekai LitRPG reskin of the Python lab manual.

You wake up as an **Unranked Pythonist** in the world of Aetherscript.  
Ten Trials stand between you and **Rank S**.  
Every trial you clear reclaims marks from the evil lab examiner **Vibhu Gautam**.

## Features
- Full progressive Status Window (Level, Rank, Focus, Caffeine, Marks, Boss HP)
- LocalStorage progress tracking
- Dynamic bars + level-up toasts
- Rank badges (F → S)
- Real Python code + verified output for every task
- Inline SVG runes + **anime-style images**
- Responsive pure HTML/CSS/JS — no build step

## Images
Decorative anime art is included in `assets/img/`:
- Crystal mage hero illustration
- Hologram / status-window style art
- Dark rune mage

## Publishing on GitHub Pages
1. Push this folder to your repo as `docs/`
2. Settings → Pages → Source: Deploy from a branch → `main` / `/docs`
3. Live at `https://<username>.github.io/<repo>/`

## Story arc
Master each Python topic → gain Rank → reclaim marks → reduce Vibhu Gautam’s HP.  
Clear all 10 → Rank S → Boss Defeated.
