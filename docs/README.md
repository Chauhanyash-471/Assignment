# Python Programming Lab — Website

A static site covering ten Python lab experiments. Open `index.html` to browse,
then click any experiment for the full write-up.

## Files

- `index.html` — landing page listing all ten experiments
- `experiment-1.html` … `experiment-10.html` — one page per experiment
- `assets/` — stylesheet and diagrams

## Publishing on GitHub Pages

1. Push this folder to your repository (a folder named `docs/` is easiest).
2. Go to **Settings → Pages**.
3. Under *Build and deployment*, set **Source** to `Deploy from a branch`.
4. Choose branch `main` and folder `/docs`, then click **Save**.
5. Wait about a minute — your site appears at
   `https://<username>.github.io/<repository>/`.

No build step and no dependencies: it is plain HTML and CSS.
